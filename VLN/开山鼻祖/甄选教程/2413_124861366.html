前言

上篇文章记录了matterport3D simulator的安装，在此基础上本篇将运行MP3D上的R2R项目。

一、升级pytorch

升级pytorch，也可以去pytorch官网 下载之前版本 

pip3 install torch==1.7.0+cu110 torchvision==0.8.1+cu110 torchaudio==0.7.0 -f https://download.pytorch.org/whl/torch_stable.html


验证安装： 

python
import torch
torch .__version__
print(torch.version.cuda)

二、安装Pytorch-Transformer 

pip3 install --upgrade pip
pip3 install sentencepiece
pip3 install pytorch-transformers

三、数据准备

1、下载 Scene features [4.2GB] (ResNet-152-Places365)到Matterport3D/img_features文件

https://link.zhihu.com/?target=https%3A//www.dropbox.com/s/85tpa6tc3enl5ud/ResNet-152-places365.zip%3Fdl%3D1

后面运行R2R可能会报错，更改文件名为ResNet-152-imagenet.tsv。

2、查看matterport3D/connectivity内容大小，也就是 connectivity maps [23.8MB]

此文件在配置验证MP3D时就已经生成，如果没有或大小不同需返回MP3D寻找问题。

3、下载 R2R data [5.8MB]

打开Matterport3D/tasks/R2R文下夹，进入确认data文件的存在，并且运行 

. download.sh

如果下载失败可以打开download，进入指定网址下载，“-p”后面是下载到的文件目录。

4、下载 collected triplets in PREVALENT [1.5GB]到Matterport3D/tasks/R2R/data文件

https://link.zhihu.com/?target=https%3A//zenodo.org/record/4437864/files/prevalent_aug.json%3Fdownload%3D1

四、运行测试

回到Matterport3D目录，运行： 

python3 tasks/R2R/train.py
